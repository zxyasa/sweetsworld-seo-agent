## 📊 潜力分数计算

**公式：** `潜力分数 = 展示量 × (1 - 点击率)`

**含义：**
- 高展示量 = 很多人在搜索
- 低点击率 = 我们的内容不够吸引人
- **机会 = 优化内容，大幅提升流量！**

**示例：**
```
关键词: "sugar free lollies australia"
展示: 2,450 次/月
点击率: 2.3%
潜力分数: 2,450 × (1 - 0.023) = 2,394

含义: 每月2,450人搜索，但只有56人点击我们的网站
如果优化到5%点击率，可以多带来 66 个访客！
```

## 🛠️ 故障排除

### 问题 1：返回"未找到足够的 GSC 数据"

**原因：**
- 网站太新，展示量不足
- 筛选条件太严格

**解决：**
```python
# 修改 seo_discovery.py 中的条件
min_impressions=50  → min_impressions=10  # 降低门槛
```

或使用主题模式：
```
"发现关于 candy 的内容机会"
```

### 问题 2：建议质量不高

**优化方法：**
- 让 OpenAI 分析更多关键词（修改 rowLimit）
- 调整 prompt 更具体描述你的需求
- 结合人工判断筛选建议

### 问题 3：GSC API 限制

Google Search Console API 有配额限制：
- 每天 1,000 次查询
- 每 100 秒 200 次

**建议：** 每天不超过 5-10 次发现操作
